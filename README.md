## About <a name="about"></a>

A ToDo Web App

### Requirements

- NodeJS
- NextJS 13
- Internet (for documentation)
- VSCode
- Firestore

### Installation

Clone this repository

```
git clone https://github.com/andeilsongf/todo.git
```

Get inside the folder project

```
cd todo
```

Install dependecies

```
npm i
```

Run project

```
npm run dev
```
